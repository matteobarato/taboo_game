# taboo_game
Simple online Taboo game written in node.js and Ionic 4 

# Launche server
Backend written in node.js based on express.js
```
cd taboo-server
npm install 
node server.js
```
server support ```rooms```, ```rooms/squad```, ```rooms/squad/player```

# Frontend
```
cd taboo-frontend
ionic serve
```
